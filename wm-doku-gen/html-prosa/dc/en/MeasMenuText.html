Offset adjustment WM3000 Performance of an offset adjustment of the complete WM3000I – without consideration of the connected components. Duration of the adjustment: approx. 15 min
Offset measurement Chanel N Performance of an offset measurements channel N:
The offset measurement channel N detects the offset in the present range with the connected components. If one component generates an offset, it will be adjusted. This process is only active if it is activated in this menu.
Offset measurement Chanel X Performance of an offset measurements channel X:
The offset measurement channel X detects the offset in the present range with the connected components. If one component generates an offset, it will be adjusted. This process is only active if it is activated in this menu.
Simulation Switches between simulation and real measurement

A session is characterized by all settings in the configuration menu and all windows selectable in “views”. All this session data will be stored “default” by leaving the software and will be set after restarting. The user can also save these settings under a special name and reactivate the session by “load”.
