Result Data File

Measuring results can be stored into result data file in XML format.

Beispiel:

The head of the XML file contains:
  • Doc type
  • Name of the measuring unit
  • Version of the measuring unit
  • Serial No. of the measuring unit
If results are stored the following data will be indicated:
  • RESULT
    • Load point in % of total RMS value and RMS of fundamental wave
    • Amplitude error in % (IEC)
    • Angle error in Â° (IEC)
    • Error according to ANSI
    • Date
    • Time
  • Conditions
    • Measuring mode in which the error has been detected.
    • Range channel N
    • Range channel X
  • Correction
  • Used correction factors of the amplitude of the standard transformer. Correction values will be derived from the used inherent error table.
