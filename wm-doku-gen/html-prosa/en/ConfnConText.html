Enter the MAC address to enable a connection between WM3000 and connected merging unit. The value 0x8100 for TPID reports the frame as IEEE802.1Q frame. The ether type 0x88BA reports the frame as layer 2 frame according to IEC61850.

Please see configuration of the connected merging unit for user priority, CFI and APPID.

You can set in which data unit the data of the transformer to be measured will be transferred inside ASDU. If you select “set”, you choose one out of eight values.
