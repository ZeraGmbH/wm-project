Geben Sie die MAC Adressen ein, um einen Verbindungsaufbau zwischen WM3000 und angeschlossener Merging unit zu ermöglichen.

Der Wert 0x8100 für TPID weist den Frame als IEEE802.1Q Frame aus. Der EtherType 0x88BA weist den Frame als Layer 2 Frame gemäß IEC61850 aus.

User priority, CFI und APPID entnehmen Sie bitte der Konfiguration der angeschlossenen Merging unit.

In ASDU stellen Sie ein in welcher Data Unit die Daten des zu messenden Wandlers übertragen werden. Mit Set wählen Sie einen von acht möglichen Werten aus.
