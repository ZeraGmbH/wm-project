Selection Description
NPrim Standard transformer primary current
NSek Standard transformer secondary current
XPrim Test primary current
XSek Test secondary current
ECTPrim Test primary current
ECTSek Test secondary voltage

Combo boxes with corresponding primary and secondary steps will be offered by entering the transformer ratio inside reference transformer errors. You can also enter the value with/without units. “No unit” will be interpreted as Ampere. You can also enter mA or kA. The measuring unit converts the values internally.
